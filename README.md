# Contact Book Application

## Description
The Contact Book Application is a Python-based project that helps users store and manage contact details easily. Users can add, search, update, view, and delete contacts through a simple and user-friendly interface.

## Features
- Store contact information such as:
  - Name
  - Phone Number
  - Email Address
  - Address
- Add new contacts
- View all saved contacts
- Search contacts by name or phone number
- Update existing contact details
- Delete contacts
- Simple and easy-to-use interface

## Technologies Used
- Python
- File Handling / Dictionary / List Concepts

## How It Works
1. User selects an option from the menu.
2. The program performs the selected operation.
3. Contact details are stored and managed efficiently.

## Operations
### Add Contact
Allows users to save a new contact with complete details.

### View Contacts
Displays all saved contacts with names and phone numbers.

### Search Contact
Search contacts using name or phone number.

### Update Contact
Modify existing contact details.

### Delete Contact
Remove unwanted contacts from the contact list.

## Requirements
- Python 3.x

## Conclusion
This project is useful for learning Python basics such as functions, loops, conditional statements, file handling, and data structures while creating a real-world application.
